# datarouter-snapshot

A Sorted String Table implementation

## Installation with Maven

```xml
<dependency>
	<groupId>io.datarouter</groupId>
	<artifactId>datarouter-snapshot</artifactId>
	<version>0.0.123</version>
</dependency>
```
