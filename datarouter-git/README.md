# datarouter-git

datarouter-git complete java based git using jgit (for git) and jsch (for ssh)

## Installation with Maven

```xml
<dependency>
	<groupId>io.datarouter</groupId>
	<artifactId>datarouter-git</artifactId>
	<version>0.0.125</version>
</dependency>
```
