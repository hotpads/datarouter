# datarouter-relay

datarouter-relay manages notification sending abstraction.

## Installation with Maven

```xml
<dependency>
	<groupId>io.datarouter</groupId>
	<artifactId>datarouter-relay</artifactId>
	<version>0.0.126</version>
</dependency>
```
