# datarouter-opencensus

## About
Instrumentation tools to interact with opencensus.io tracing. 

## Installation with Maven

```xml
<dependency>
	<groupId>io.datarouter</groupId>
	<artifactId>datarouter-opencensus</artifactId>
	<version>0.0.111</version>
</dependency>
```
