# datarouter-email
## About
Instrumentation and tools for sending emails

## Installation with Maven

```xml
<dependency>
	<groupId>io.datarouter</groupId>
	<artifactId>datarouter-email</artifactId>
	<version>0.0.115</version>
</dependency>
```
