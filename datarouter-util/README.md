# datarouter-util

datarouter-util is a set of utility classes.


## Installation with Maven

```xml
<dependency>
	<groupId>io.datarouter</groupId>
	<artifactId>datarouter-util</artifactId>
	<version>0.0.45</version>
</dependency>
```
