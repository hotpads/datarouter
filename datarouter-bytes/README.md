# datarouter-bytes

datarouter-bytes is a set of utility classes.

## Installation with Maven

```xml
<dependency>
	<groupId>io.datarouter</groupId>
	<artifactId>datarouter-bytes</artifactId>
	<version>0.0.97</version>
</dependency>
```
